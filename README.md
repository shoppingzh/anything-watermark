# anything-watermark
万物皆可水印





### 水印设置项
- 类型：文字、图片
- 坐标：(x, y) 坐标值为浮点数则按照比例计算，否则按照实际像素计算
- 对齐方式：x轴始、终对齐与y轴始、终对齐，因为水印具有宽高，该参数实际用于确定水印中心点的位置
- 大小：相对其容器的大小，是一个比例值，取值区间为(0 , 1]
- 旋转角度：[0 - 360]
- 透明度：[0 - 1]
- 是否遮挡内容：设置后将有可能遮挡主体信息
- 【文字水印】字体类型、字体大小
- 【文字水印】颜色
- 【文字水印】字体样式（加粗、斜体、下划线、中划线）
- 【图片水印】图片文件
